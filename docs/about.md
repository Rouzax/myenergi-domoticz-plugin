## Acknowledgements

The myenergi cloud API has no public documentation. The field meanings this plugin relies on
(status codes, plug states, energy fields, and so on) were cross-checked against two community
projects:

- [twonk/MyEnergi-App-Api](https://github.com/twonk/MyEnergi-App-Api), a reverse-engineered
  reference for the myenergi app's API.
- [CJNE/pymyenergi](https://github.com/CJNE/pymyenergi), an async Python library for myenergi
  devices.

Thanks to their authors and contributors for documenting a protocol myenergi itself does not
publish.
